# RESTOCK

RESTOCK is a personal Home Assistant experiment for tracking inventory
containers with NFC/RFID tags and an ESPHome M5Dial scanner.

## Experimental Status

RESTOCK is shared as-is. It is not a polished or official Home Assistant
integration, and it should be used for testing and learning rather than relied
on for critical inventory tracking.

## Install

Install through HACS as a custom integration repository:

```text
https://github.com/leseulsteve/resto-hacs
```

The ESPHome M5Dial config is included in `esphome/m5dial/`.
